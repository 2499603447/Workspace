**should和shouldn't**
情态动词should提出建议和忠告;情态动词否定式shouldn't建议某人不要做某事
* 主语+should+主要动词
You should go./We shouldn't go
* 组成一般疑问句，将should置于句首
Should we go
* 组成特殊疑问句，先使用疑问词
What should we do this weekend?/Where should we go?
